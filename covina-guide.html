System Architecture & Directory Layout

Medi Tech Data DMS stores medical PDFs directly within structured local folders (DB/). Patient directory paths follow the pattern below:

DB / [LAST]_[FIRST]_[MIDDLE]_[DOB] / [Category Subfolder] /

Each patient folder contains the following standard category subfolders:

Category Channel Target Storage Subfolder Description
Progress Notes…/Progress Notes/Clinical evaluation notes & progress summaries
Insurance…/Insurance/Insurance cards & policy documents
Authorization…/Authorization/Procedure authorization approvals
Eligibility…/Eligibility/Coverage verification & eligibility forms
Lab Reports…/Lab Reports/Diagnostic lab tests & bloodwork records
Radiography Reports…/Radiography Reports/X-Ray, MRI, CT scans & imaging reports

Searching & Selecting Patients

Follow these steps to find a patient record and load their profile:

  1. Select Search & Patient Profiles from the sidebar menu.
  2. Enter search keywords in the Search Filter Panel — First Name, Last Name, Middle Name, or Date of Birth.
  3. Entering digits in the DOB field automatically formats the value to MM-DD-YYYY.
  4. Click Select Active Patient Profile to load the full patient record view.
Tip: You can search by any single field — you don't need to fill in all four. A last name alone is often enough to narrow results.

Uploading Documents to Categories

To add new PDF files to a specific category folder for an active patient:

  1. Select the active patient profile as shown in Section 2.
  2. Click the category tab you want to upload to (e.g., Authorization).
  3. In the left pane (Add New Document), drag and drop your PDF file or click Browse files.
  4. The system stores the document directly into that category folder and refreshes the records index.
Important: Only PDF files are accepted. Uploading other file types (e.g., DOCX, JPG) will not be processed by the system.

Downloading Archived Documents

Option A — Download from Individual Patient Profile

  1. Select the patient's active category tab (e.g., Lab Reports).
  2. Under Stored Records Index on the right pane, pick the file name from the list.
  3. Click Download to save the file locally.

Option B — Download via Bulk Category Tools

  1. Switch sidebar navigation to Bulk Category Repository Tools.
  2. Choose the target document category from the dropdown menu.
  3. Select the file from the master list and click Download Global File.
When to use Bulk Tools: Use Option B when you need to retrieve multiple files across different patients in the same category, rather than opening each patient profile individually.
5

Safely Exiting the Application

Always use the built-in exit control to shut down the application cleanly when you are finished:

  1. Click the red Exit System button in the top-right header bar.
  2. This closes all active browser tabs and stops the background Python process.
Do not close the browser tab directly without clicking Exit System — the background process will remain running until the system is restarted.
